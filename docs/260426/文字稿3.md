最后说下这个页面提供的“验证”功能。
在搜索到的每个数列的右上角，都有一个验证按钮。
点击就能看到对话框，直观地验证这个数列确实符合题意。
并且，我们还做了一个“自定义数列验证”组件。
输入一个数列，就能直观地验证它确实符合题意。
