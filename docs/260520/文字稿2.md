这里我们先讲怎么用概率生成函数做这题，
再讲这题的不超纲做法，
看到最后，相信你会理解为什么我要特意调换讲解顺序！
设X是离散型随机变量且X为自然数，
则X的概率生成函数定义为屏幕里的1式。
首先，初步观察这条定义式，
不难发现G（1）=1，因为是所有概率之和。
接下来我们尝试求导，看看能发现什么性质！
求1阶导发现，G撇1就是X的数学期望。
再求2阶导发现，G两撇1就是X乘X减1的数学期望。
如果你觉得规律还不明朗，
可以再求更高阶导，看看效果。
总之，我们发现，
G的m阶导在1处的值为A X m的数学期望。
这里A X m就是我们都很熟悉的排列数，
但在OI里，它还有另一个名字——下降幂。
请记住这个词，后续我们会经常见到它。

为了用上上一页得到的性质，
我们便希望用下降幂的数学期望解出这道题。
接下来该怎么做呢？
不难发现，A X m是关于X的m次多项式，
并且A X 1、A X 2、……、A X m是线性无关的，
所以我们猜想，X的m次方应该是可以用
这m个下降幂的线性组合表示的。
这个想法对这道题有什么启发呢？
诶，根据期望的可加性，我们只需要求出
A X 1、A X 2、A X 3的数学期望，
就能用它们的线性组合表示出这题想要的X的3次方的数学期望。
但问题来了，怎么确定线性组合的系数呢？
待定系数法自然能做，但如果你打过OI，
就会知道有一个公式可以更方便地求出这些系数，
那就是下降幂转普通幂公式。
它告诉我们，下降幂的线性组合的系数就是第二类斯特林数。
这条公式的推导不是本期视频的重点，
我们先不讲，后续有时间再做视频讲解了~
这里直接给出我们想要的线性组合的系数：1,3,1。
至此，我们就成功把原问题转化为
求X的下降幂的数学期望了。

我们已经知道下降幂的数学期望
就是概率生成函数的m阶导在1处的值。
但我们还不知道二项分布的PGF呢，又怎么求m阶导呢！
所以下面我们就来推导二项分布的PGF。
设X服从二项分布，q=1减p，
把成功k次的概率代入概率生成函数的定义式，
我们发现p的k次方和z的k次方刚好可以合并成p乘z的k次方，
由二项式定理，我们成功合并出一个简洁、优美的式子。
但如果注意力和汉斯一样涣散怎么办？
我们不妨用概率生成函数的乘积性质，
给出二项分布的PGF的另一种推导方法。
设随机变量X和Y独立，则X+Y的概率生成函数，
就是它们的概率生成函数的乘积。
这个性质可以自然地推广到多个独立随机变量的情况，
这里不赘述。

下面简单证明下刚刚说的性质。
由X、Y独立可以写出X+Y=k的概率。
如果你熟悉多项式乘法，就很容易发现，
这条式子和多项式乘法的定义式可太像了！
我们逆用多项式乘法的定义式，
就成功证明了这条性质。
接下来用这条性质推导二项分布的PGF。
在高中我们知道，0 1分布是做1次伯努利试验，
二项分布是做n次独立的伯努利试验。
设有n个服从0 1分布的随机变量，
则X=X1加到X n服从二项分布。
由PGF的乘积性质可知，
求出X i的PGF，再乘起来就是所求。
由PGF的定义式，很容易求出0 1分布的PGF就是q+pz。
所以二项分布的PGF就是q+pz的n次方。
幸运的是，二项分布的PGF的m阶导很容易求，
所以预计这题的计算量不会太大。

我们已经求出了二项分布的PGF，
也知道怎么用下降幂表示X的3次方的数学期望，
还知道怎么求出下降幂的数学期望。
现在我们可以轻松做出这题了！
代入n=20，p=0.5可得这题的具体PGF，
逐次求导，不难算出：
X的数学期望是10，
A X 2的数学期望是95，
A X 3的数学期望是855，
所以答案就是855+3*95+10=1150。

下面我们总结一下今天学到的芝士。
首先是PGF的优美性质。
1，G（1）=1。
2，G的m阶导在1处的值为A X m的数学期望。
据此，可以用A X m的数学期望的线性组合
表示一切X的多项式的数学期望。
3，PGF的乘积性质。
我们还推导了常见离散分布的PGF。
0 1分布的PGF是q+pz。
因为0 1分布是做1次伯努利试验，
二项分布是做n次独立的伯努利试验，
所以二项分布的PGF就是0 1分布的PGF的n次方。

为了用A X m的数学期望的线性组合
表示一切X的多项式的数学期望，
我们还需要用到下降幂转普通幂公式。
有了这些芝士，我们还可以尝试做一些更难的题。
1，把二项分布的参数变大，
并约定模998244353，
这就是一道算法竞赛题了。

常规题解就是用第二类斯特林数和组合恒等式各种推，

最后我们来讲一种不超纲的做法。
从期望的定义出发，
我们可以直接写出屏幕上方的1式。
我们观察1式，发现p刚好等于q，
p的幂乘q的幂就是2的20次方分之1，
于是问题简化为对k的3次方乘C n k求和。
但简化后的式子依旧不好算，怎么办呢？
回想前面对PGF求m阶导时，
我们发现对幂函数多次求导，
下降幂A k m自然就会出现。
受这一观察启发，我们把普通幂k的3次方
拆分为下降幂的线性组合，
于是我们发现，问题被进一步转化为
对A k m乘C n k求和，
不妨设这个求和式为函数g（m），
在这题里，m只取1、2、3。

那如何让g（m）的系数A k m出现呢？
这就是高中很常见的套路了：二项式定理+求导！
我们对1+x的n次方求高阶导，看看效果。
不出所料，系数A k m出现了，
所以f的m阶导在1处的值就是我们想要的g（m）。
于是用f的1、2、3阶导就能求出答案了。
