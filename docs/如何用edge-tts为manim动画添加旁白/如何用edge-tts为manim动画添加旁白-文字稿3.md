个人用edge-tts给manim动画添加旁白的方案比较朴素
首先生成音频和字幕文件
接着根据字幕文件的时间轴来确定manim动画的播放时间
重新生成manim的mp4
这里设置的播放时间要比旁白阅读时间长一些
以屏幕里展示的代码为例
我们查看srt，确定好旁白的阅读时长后
手动修改manim代码的wait time
而各种manim动画的run time都保持默认就行
最后用剪辑软件剪掉多余的播放时间
