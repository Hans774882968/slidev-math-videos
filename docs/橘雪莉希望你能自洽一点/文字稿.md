宝，今天是什么日子呀？
你是不是又在酸别人的朋友圈九宫格？
你是不是又在群里发各种抽象文案和梗图？
你是不是连发个“今天好累”都要纠结，
这算不算不够抽象，不合群？
但你不觉得这一切很诡异吗？
这不过是一串符号而已，
却能每三百六十五个地球日，
准时给你的情绪上发条：
该甜了，该晒了，该抽象了。

左边是“别人家的浪漫”实时投喂，
右边是“你该幸福了”的无形期待。
连emo都要包装成“思想深刻”的人设。

宝，你想呀，
我们随时可以跳出这一切的！
嗯？
宝，你说人无往不在竞争之中，
赢感也是人的基本需要？
可我不想再看到你为了各种隐形排名牵着走啦！
我真的好希望你能活得轻盈一点！
嗯？
宝，你说我希望你自洽一点，
不也是在用一串符号规定你的情绪和行为吗？
恭喜你拥有了最重要的东西——
反思、追问的能力。
你当然可以继续为别人的晒图而心里咯噔！
你当然可以继续为文案不够抽象而懊恼！
你当然可以继续为没打到全良连段而惋惜！

但我的宝，
我只是希望你既能坦然享受赢感，
又能在被“必须幸福”的KPI压得难受的时候，
轻轻追问那么一句，
这情绪，是我想要的，还是被安排的？
