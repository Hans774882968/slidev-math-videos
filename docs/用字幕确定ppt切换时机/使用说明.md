## 引言

如果输入的是edge-tts生成的字幕，那么效果会很好，因为edge-tts会比较忠实地用文字稿创建字幕（偶尔会吞掉末尾的标点符号）

如果输入的是字幕识别生成的字幕，那么效果一般会比较差。这时建议手工创建`patbs-idx.md`，会浪费点时间，但不再需要提供文字稿文件

## 使用方式示例

只在终端输出字符串：

```powershell
bun autoslide -s media\260507\文字稿2.srt -t docs\260507\文字稿2.md -sc video-blogs\subtitle_cfg\分式线性递推数列的高中和大学解法.json -p
```

修改 ppt 文件：

```powershell
bun autoslide -s media\260507\文字稿2.srt -t docs\260507\文字稿2.md -sc video-blogs\subtitle_cfg\分式线性递推数列的高中和大学解法.json -r video-blogs\【slidev】分式线性递推数列的高中和大学解法.md

bun autoslide -s media\260508\文字稿2.srt -t docs\260508\文字稿2.md -sc video-blogs\subtitle_cfg\轻松一刻-排列组合网红题.json -r video-blogs\【slidev】轻松一刻-排列组合网红题.md
```

手工创建索引文件并输入：

```powershell
# 修改 ppt 文件
bun autoslide -s media\260508\文字稿2.srt -i docs\260508\patbs-idx.md -r video-blogs\【slidev】轻松一刻-排列组合网红题.md
# 只在终端输出字符串
bun autoslide -s media\260508\文字稿2.srt -i docs\260508\patbs-idx.md -p
```
