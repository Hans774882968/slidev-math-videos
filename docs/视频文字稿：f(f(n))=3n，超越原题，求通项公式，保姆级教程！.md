选项A显然
选项B可以用反证法，设a n 小于等于n，很容易推出矛盾
选项C，从a1推到a5就行
由B选项，a1只能是1和2，但1不行，所以只能是2
把a a1代入约束方程，就拿到a2
然后a2代入约束方程，就拿到a3已经是6
所以C不对
选项D，注意到2025和25只差3的幂
所以我们希望得到a 3n和a n的关系
为了方便理解，我们换个元，然后嵌套3层，就拿到a 3n=3a n
这题也就做完了
简单总结一下选项D，我们用了两个不同视角去看3层嵌套
一个2层+1层，另一个1层+2层
这种通过多个视角拿到隐含方程的思想在数学中常常出现

《一试》那题不需要求通项公式就能做
所以现在我们直接讲怎么求通项公式
直接求通项公式没头绪啊！
我们不妨先求出前几项，看看规律
想亲自尝试的同学不妨暂停视频手推一下
接下来我们开始讲手推方法
我们现在有1到3的值，1和2都多嵌套过一层了
那么由排除法可知，只剩一种操作
就是对a3再套一层，得到a6=9
这时我们发现9减6=6减3，而且数列严格单增
所以4和5就被夹住了，值分别是7和8
对a4和a5再套一层，就拿到a7和a8
由a3可以求出a9
之后的10到17、19到26都可以类似地算出来
在此不再赘述

屏幕给了一张表，不难总结出规律
1到2倍3的k次方范围内，每次增长1
2到3倍3的k次方范围内，每次增长3
顺便，《一试》那题答案是83

我们看表说话，不难写出通项公式
接下来我们开始证明
首先是n是3的幂的情况
我们知道a1和a 3n=3a n就做完了
接着是1到2倍3的k次方范围的式子
其实在前面手算的时候就隐隐约约感觉到推法了
对a 3的k次方多套一层，就得到
a 2乘3的k次方=3的k+1次方
我们发现它减去a 3的k次方等于两者的下标差，3的k次方
但这是严格单增数列，所以中间的每一项都被夹住了
最后是2到3倍3的k次方范围的式子
这个比前两个稍难，需要一丢丢注意力
我们回顾前面手推的过程，发现是用情况2推出情况3的
这就启发我们发现，n减去3的k次方，就变成了情况2
并且情况2的通项公式刚好加回n
所以我们设出屏幕里的t，然后多嵌套一层，就拿到a n啦
