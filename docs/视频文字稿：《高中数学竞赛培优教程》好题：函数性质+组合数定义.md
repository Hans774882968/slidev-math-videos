题目给了一个抽象函数满足的一些性质
求这个式子的值
这种题的思路简单粗暴
根据目标函数的长相，不断地手玩约束方程
直到找到能用来化简目标函数的性质

为了书写方便，下面记目标函数为g
看到目标函数里有x分之1
我们自然会想把它代入约束方程试试
得到屏幕里的式子
单看这个式子没啥用，但没关系
我们看到f是奇函数，而x分之1也是奇函数
所以我们会想把上面的x换成负x试试
我们得到了一个连等式！
就是屏幕里框起来的式子

这个连等式只对负数x成立，但目标函数里都是正数
所以我们把负x代入这个连等式
把减x写成加u
我们就拿到了一个递推式！
结合f1=1，我们就拿到了f x分之1的通项公式
于是目标函数可以写成屏幕里带有两个阶乘的乘积的形式

这里就不可避免要用到一点点注意力了
i的阶乘和99减i的阶乘让我们联想到组合数的定义式
于是我们拿到了这个带有组合数的前缀和的式子
看到这个前缀和要前50项
而这一行杨辉三角刚好有100项，50的两倍
所以我们自然会想到用二项式定理来算
它就是2的99次方的一半
于是我们就拿到答案啦
所以只有这一页PPT的过程需要一丢丢注意力
